# 预览体验计划
CDA.net 现已迈向全新的版本！为了测试、实验新功能，并获取用户反馈，CDA.net 制定了 CDA.net Insider Preview 预览体验计划。<br>
CDA.net 预览体验计划独立于正式版本的 CDA.net，使用独立的域名来和正式版本做区分。<br>
请注意：CDA.net Insider Preview 预览体验计划中的一些功能，可能还未制作完成，并且 CDA.net 相较于正式版本，可能遇到的问题通常更多，有些时候甚至根本无法使用基础功能（例如一次完全重做的大更新的预览）。<br>
## 开始
你当前已正在使用 CDA.net Insider Preview 预览体验计划门户。
[点击此处](https://miner233.github.io/cda) 以切换到 CDA.net 正式版。<br>
