    CDA.net 本地化计划¶

    为 CDA.net 带来一门新的语言！

    如何参与¶

    进入 CDA.net Insider Preview Github 存储库，Fork 这个存储库的 main 分支（语言文件存储于 main 分支），随后使用 git clone 命令 1 将你 Fork 的存储库 clone 下来。
    clone 下来后，使用一款文本编辑器（推荐 Visual Studio Code！真的很好用！） 2 打开语言文件 3，并开始编辑。

    编辑完后，将更改推送到你 Fork 的存储库中，并打开原存储库（https://github.com/Miner233/cda_pre），发布拉取请求（Pull request），等待我们审核并答复。

    1. 请确保你的电脑已有 Git 环境。 ↩

    2. 大部分文本编辑器均可。打开时请使用 UTF-8 编码，否则可能会出现乱码。 ↩

    3. 语言文件，xx-xx.json，像 zh-cn.json 就是简体中文语言的语言文件，en-us 就是 English 语言的语言文件。语言文件存储在 GitHub 存储库 main 分支的 language 目录下。可能没有部分语言的语言文件，可以联系我们添加（发布 issues）。 ↩
